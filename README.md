## A simple python application to sync your medium blogs

Lets introduce the CI/CD to our medium blogging

## Prerequistes
1. Download python
1. Download dependencies
```bash
    pip install -r requirements.txt
```
## Running the script
```bash
export MEDIUM_API_KEY=<your-api-key>
python3 main.py

```
